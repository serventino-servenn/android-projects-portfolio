## Android-projects-portfolio

A collection of Android projects showcasing my journey learning Kotlin, Jetpack Compose, and modern Android app architecture through hands-on, independent implementation.
The projects are arranged from **beginner to intermediate/advanced**, reflecting my growth in UI design, state management, networking, local persistence, and adaptive layouts.


## Tech Stack

**Language:** Kotlin

**UI:** Jetpack Compose, Material 3

**Architecture:** MVVM, Repository Pattern

**Async:** Coroutines, StateFlow

**Persistence:** Room, DataStore

**Networking:** Retrofit, Gson

**Images:** Coil

**Testing:** Unit and instrumented tests

**Tools:** Android Studio, Git, GitHub
